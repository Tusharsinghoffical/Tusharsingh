Project Overview

This project is a Python-based AI Study Assistant that utilizes the Gemini API to generate structured notes, multiple-choice questions (MCQs), and explanations for any given topic. It features a user-friendly GUI built with Tkinter, making it easy to input topics and receive comprehensive study materials.
